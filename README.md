# InstaCrawlerBot
Uma aplicação feita para auxiliar em uma disciplina na graduação de História - UFOP

## Análise de Tweets: AOC vs Trump

Este repositório contém uma análise comparativa detalhada dos estilos de postagem de Alexandria Ocasio-Cortez (AOC) e Donald Trump no Twitter/X.

### Recursos Disponíveis

- **[data](https://github.com/ayrtondenner/InstaCrawlerBot/tree/copilot/create-report-analysis-aoc-trump/data)**: Pasta contendo os tweets raspados dos políticos AOC e Trump em formato JSON
- **[analysis.ipynb](https://github.com/ayrtondenner/InstaCrawlerBot/blob/copilot/create-report-analysis-aoc-trump/analysis.ipynb)**: Notebook Jupyter com análises estatísticas detalhadas dos tweets coletados
- **Relatórios de Análise:**
  - [Relatório em Português (PT-BR)](https://github.com/ayrtondenner/InstaCrawlerBot/blob/copilot/create-report-analysis-aoc-trump/relatorio_analise_tweets.md)
  - [Report in English (EN-US)](https://github.com/ayrtondenner/InstaCrawlerBot/blob/copilot/create-report-analysis-aoc-trump/twitter_analysis_report.md)

Os relatórios incluem análises sobre:
- Volume e distribuição das postagens
- Estilo e características do conteúdo
- Temas e assuntos abordados
- Métricas de engajamento e correlações estatísticas
- Análise comparativa entre os dois políticos
